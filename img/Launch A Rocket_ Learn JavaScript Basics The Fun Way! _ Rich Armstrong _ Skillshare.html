      Project Description
      
                
      
                    
      Build a rocket launch sequence with JavaScript, HTML and CSS
      

      

      After learning the basics of JavaScript, we’ll plan our rocket launch sequence; build the basics with HTML and CSS; add the JavaScript to make in interactive; and then finish it off by making it look amazing with some more CSS! End it off by showing off and sharing it with the other students in the project gallery :)
      

      ASSIGNMENT
      

      Create a rocket launch sequence with JavaScript, HTML, and CSS – and then share it with the other students in the class! The assignment will give you a structured environment to learn, play and test things out. You’ll learn what all the slashes, dots, braces, brackets and funny words mean. And you’ll also learn about some advanced CSS properties and CSS transitions!!

      

      DELIVERABLES
      

      - A launch button.
      - A CSS styled rocket.
      - A rocket that launches upwards.
      - A countdown sequence.
      - A programmed random chance of failure.
      - Sharing your project in the project gallery.
      

      If you're keen, read on!
